<p>振ったダイス目の合計値をランダムに出すツールです。</p>

<h2>【操作手順】</h2>
<ul>
 <li>ダイスの個数と面数の値の両方を設定したあと、「結果を表示」ボタンを押してください。合計値が表示されます。</li>
 <li>「リセット」ボタンを押すと、ダイスの個数と面数をリセットできます。</li>
</ul>

<h2>【アピールポイント】</h2>
<ul>
  <li>ダイスの個数・面数の両方に数字が入力されていない状態でボタンを押すと、アラートが表示されるようにしました。</li>
  <li>リセットボタンを実装しました。</li>
</ul>

<h2>【改善点】</h2>
<ul>
  <li>それぞれのサイコロの目も記述しようとしましたが、2回目以降の結果を非表示にする方法がわからず断念しています。</li>
  <li>「もう一度ダイスを振る」ボタンがあるとより分かりやすいと思います。</li>
  <li>結果表示欄に枠線を入れた方が視認性がよさそうです。</li>
</ul>
